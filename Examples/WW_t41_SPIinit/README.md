 "Experiment_SDIO2" branch of his SdFat repo from here:

using uSDHC2 (SDIO2). Pin connections:

T41                     WIFI

22 (CMD)           CMD

23 (CLK)            CLK

40 (D0)              D0

41 (D1)              D1

17 (D2)              D2

16 (D3)              D3

3.3V                   3.3V

GND                  GND

using 3" jumper wires at the moment.
